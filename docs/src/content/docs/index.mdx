# Test Generation for Prompts

**Prompts** are an important part of any software project that incorporates
the power of AI models. As a result, tools to help developers create and maintain
effective prompts are increasingly important.

**PromptPex** is a tool for exploring and testing AI model prompts. PromptPex is
intended to be used by developers who have prompts as part of their code base.
PromptPex treats a prompt as a function and automatically generates test inputs
to the function to support unit testing.

## Features

<CardGrid stagger>
  <Card title="Automated Test Generation" icon="open-book">
    PromptPex uses [an LLM-based process](/promptpex/reference/) to generate a set of inputs that capture the behavior of your prompt.
  </Card>
  <Card title="Output Rules Evaluation" icon="approve-check-circle">
    Eval your prompt's output compliance against a set of rules that are automatically extracted from your prompt,
    or define your own [metrics](/promptpex/reference/metrics).
  </Card>
  <Card title="Export to OpenAI Evals" icon="add-document">
    Export generated tests and metrics using (Azure) [OpenAI Evals](/promptpex/reference/evals).
  </Card>
  <Card title="Azure OpenAI Store Completions" icon="cloud-download">
    Use generated tests to distillate to smaller models using [Azure OpenAI Stored Completions](/promptpex/reference/stored-completions).
  </Card>
  <Card title="Bring your own LLM library" icon="approve-check-circle">  
    Integrate with your own LLM library or use our [GenAIScript](/promptpex/dev/genaiscript) / [Python](/promptpex/dev/python) implementations
  </Card>
  <Card title="Flexible Generation" icon="setting">
    Use [scenarios](/promptpex/reference/scenarios), [test expansions](/promptpex/reference/test-expansion) or [custom instruction](/promptpex/reference/custom-instructions) to configure the test generation.
  </Card>
</CardGrid>
