Scenarios

PromptPex supports specify a set of additional input constraints (scenario) to generate specific test suites. A canonical example would be localization testing: generate English, generate French.

PromptPex enumerates through the scenarios and the rules and generates a test for each combination.

Configuration

Section titled “Configuration”

The test generation scenarios are configured in the prompt front-matter. You can decide to fill in any of the template variables in each scenario.
