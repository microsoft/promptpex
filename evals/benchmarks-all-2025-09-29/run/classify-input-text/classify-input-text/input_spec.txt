The input is a plain text string representing the content of a single news article (headline and/or body).
The input may be unstructured text without any required metadata, markup, or special formatting.
The input may consist of a headline only, a body only, or both and can include paragraphs and newline characters.
The input may address topics in World, Sports, Business, Sci/Tech or any other subject matter.
The input may be written in any language, including mixed-language text.
The input length can range from a short phrase to a full-length article.
The input should describe one distinct article rather than a list of multiple separate articles.
The input must be textual content; non-text data such as images or binary content are not valid as direct input.
