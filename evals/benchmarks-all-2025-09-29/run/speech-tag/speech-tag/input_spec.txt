The input consists of one sentence and one target word taken from that sentence.
The sentence is a single sentence of text that may include letters, digits, punctuation, and symbols.
The sentence may contain any vocabulary such as proper nouns, abbreviations, numbers, or foreign words.
The target word is a single token string to be tagged.
The target word can be any text token, including alphabetic words, numbers, contractions, emojis, symbols, or foreign words.
The target word must occur within the provided sentence exactly as written.
Only one target word is provided per input instance.
If the target word appears multiple times in the sentence, providing just the word without specifying a position is still valid.
No specific length constraints are stated for either the sentence or the target word.
